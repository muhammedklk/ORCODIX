## ↳ About ORCODIX

**ORCODIX** is a high-end, award-winning design and creative development studio. Founded on the principle of **"Design and Code As One,"** we eliminate the traditional friction between designers and developers to create fluid, pixel-perfect, and ultra-high-performance web applications that leave a lasting impression.

---

## ⚡ Core Pillars

*   ✨ **Precision Craftsmanship**: Hand-crafted layouts, clean codebases, and mathematically precise alignment.
*   🧭 **Immersive Interactions**: Rich micro-animations, tactile spring physics, custom page transitions, and responsive design systems.
*   🛡️ **Strategic Impact**: Conversion-focused architecture built around business objectives to convert visitors into advocates.

---

## 🛠️ Technology Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/) & Vanilla CSS variables for theming
*   **Animation**: [Framer Motion](https://www.framer.com/motion/) for physics-based animations
*   **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

To run the project locally, follow these simple steps:

### 1. Clone the repository
```bash
git clone git@github.com:muhammedklk/ORCODIX.git
cd ORCODIX
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the live application.
